# mevnproject
this a test projet build on top of MVEN (MongoDB, Vue, Express, Node) stack,

## Project setup
1 - first of all clone the project : [here]() <br/>
2 - go into the project folder <br/>
3 - install dependencies

### here are the command for that

```
git clone

npm install
```
